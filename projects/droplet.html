Introduction

n today’s world, automation plays a crucial role in simplifying everyday tasks. One such task is managing water systems, whether for farms, gardens, or home setups. This project introduces a smart solution by integrating an Arduino Nano with GSM communication to control a water pump remotely. Paired with a Flutter-based mobile app, this system provides real-time control, status updates, and scheduling capabilities to manage the pump effortlessly.

Features

Remote Pump Control via GSM

The core functionality of this project is its ability to control the water pump from anywhere using GSM technology. Through the Droplet app, users can send commands via SMS to start or stop the pump remotely, providing convenience and control from any location.

Timer-Based Pump Scheduling

Users can schedule the pump’s operation by setting a timer within the app. The app ensures the pump runs for a predefined duration, after which it automatically stops. This feature is particularly useful in scenarios where precise water management is required, such as irrigation or timed water supply.

Pump Operation History

Every command sent and every pump operation is logged in the app. This historical record allows users to review when the pump was started, stopped, or if any issues occurred. This feature offers valuable insights into the system's performance and can help with long-term water management planning.

Mode Night

The app includes a dark mode feature, designed to reduce eye strain during late-night work sessions while maintaining a sleek and modern design. Whether you're browsing during the day or the evening, the night mode ensures a clean, minimalistic look that adapts to any time of day.
